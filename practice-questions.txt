Find all lines containing ERROR.

Solution:  awk '$3=="ERROR" {print}' chatgptlog.log

2025-09-18 10:16:02 ERROR Database connection failed
2025-09-18 10:20:12 ERROR File not found: /var/www/html/index.html

Count how many WARN messages are in the log.

Solution:
 awk '$3=="WARN" {count++} END {print count}' chatgptlog.log
2
